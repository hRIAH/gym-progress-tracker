# Gym Progress Tracker 💪

A simple Java console application to track workout progress.

## 📌 Features
- Add exercise name
- Track weight lifted (kg)
- Track number of reps

## 🛠️ Tech Stack
- Java
- Scanner (for input)

## 🚀 How to Run
1. Install Java (JDK)
2. Compile:
   javac GymTracker.java
3. Run:
   java GymTracker

## 📷 Sample Output
Enter exercise name:
Bench Press

Enter weight (kg):
80

Enter reps:
10

--- Workout Log ---
Exercise: Bench Press
Weight: 80 kg
Reps: 10
